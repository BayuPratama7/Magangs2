SISTEM PENGELOLAAN MAGANG PRODI SISTEM INFORMASI

ROLE:
1.PENGELOLA PRODI= KETUA PROGRAM STUDI 
KOORDINATOR PENGELOLA MAGANG, SEKRETARIS PROGRAM STUDI
2.DOSEN PEMBIMBING LAPANGAN
3.MAHASISWA
4.PENGUJI DESIMINASI

KETUA PROGRAM STUDI MEMILIKI HAK AKSES SEBAGAI BERIKUT:
1. ACC KEDUA SETELAH ADANYA ACC DARI KOORDINATOR PENGELOLA MAGANG

KOORDINATOR PENGELOLA MAGANG MEMILIKI HAK AKSES SEBAGAI BERIKUT:
1. ACC PERTAMA SETELAH ADANYA INPUT LINK DRIVE PROPOSAL
2. MELIHAT LINK DRIVE LOGBOOK MAHASISWA
3. MELIHAT HASIL ACC LAPORAN HASIL MAGANG AKHIR DARI PENGUJI DESIMINASI

SEKRETARIS PROGRAM STUDI MEMILIKI HAK AKSES SEBAGAI BERIKUT:
1.INPUT MENENTUKAN DOSEN PEMBIMBING LAPANGAN MAGANG UNTUK MAHASISWA SETELAH ADANYA ACC DARI KETUA PROGRAM STUDI
2.INPUT SURAT PENGANTAR MAGANG
3.INPUT MENENTUKAN PENGUJI DESIMINASI
4.INPUT JADWAL DESIMINASI
5.INPUT KONTEN DASHBOARD INFORMASI MITRA KERJASAMA MAGANG PROGRAM STUDI SISTEM INFORMASI
6.INPUT KONTEN DASHBOARD INFORMASI SEBARAN MAGANG BERDASARKAN WILAYAH DI PERIODE SEBELUMNYA
7.INPUT KONTEN DASHBOARD INFORMASI SEBARAN MAGANG BERDASARKAN JENIS MAGANG (REGULER,BUMN,MBKM) DI PERIODE SEBELUMNYA

DOSEN PEMBIMBING LAPANGAN MEMILIKI HAK AKSES SEBAGAI BERIKUT:
1.MELIHAT DAFTAR MAHASISWA SESUAI YANG DITENTUKAN SEKRETARIS PROGRAM STUDI
2.MELIHAT LINK DRIVE LOGBOOK MAHASISWA
3.REVIEW HASIL LAPORAN MAGANG MAHASISWA SEBELUM MENDAFTAR DESIMINASI DAN ACC
4.MELIHAT JADWAL DESIMINASI MAHASISWA
5.MELIHAT STATUS MAHASISWA SUDAH DESIMINASI ATAU BELUM

MAHASISWA MEMILIKI HAK AKSES SEBAGAI BERIKUT:
1.INPUT LINK DRIVE PROPOSAL MAGANG
2.MENDAPATKAN SURAT PENGANTAR MAGANG DARI SEKRETARIS PRODI
3.MENDAPATKAN DOSEN PENGUJI LAPANGAN
4.INPUT LINK DRIVE LOGBOOK KE DOSEN PEMBIMBING LAPANGAN DAN KOORDINATOR PENGELOLA MAGANG
5.INPUT LINK DRIVE LAPORAN HASIL MAGANG KE DOSEN PEMBIMBING LAPANGAN
6.FIELD TAMBAHAN BERSIFAT (OPSIONAL) BERISI SURAT PERNYATAAN DITERIMA MAGANG
7.MELIHAT DASHBOARD INFORMASI MITRA KERJASAMA MAGANG PROGRAM STUDI SISTEM INFORMASI
8.MELIHAT DASHBOARD INFORMASI SEBARAN MAGANG BERDASARKAN WILAYAH DI PERIODE SEBELUMNYA
9.MELIHAT DASHBOARD INFORMASI SEBARAN MAGANG BERDASARKAN JENIS MAGANG (REGULER,BUMN,MBKM) DI PERIODE SEBELUMNYA
10.PENGAJUAN DESIMINASI DENGAN INPUT LAPORAN HASIL MAGANG BERDASARKAN ACC DARI DOSEN PEMBIMBING LAPANGAN

PENGUJI DESIMINASI MEMILIKI HAK AKSES SEBAGAI BERIKUT:
1. ACC UNTUK MENGUJI MAHASISWA YANG SUDAH DITENTUKAN OLEH SEKRETARIS PROGRAM STUDI
2. ACC LAPORAN HASIL MAGANG AKHIR YANG SUDAH DI REVISI DARI MAHASISWA


ALUR PROSES BISNIS MAHASISWA DI SISTEM INI:
1.MAHASISWA MENGAJUKAN/INPUT LINK DRIVE PROPOSAL MAGANG UNTUK DIAJUKAN KE KOORDINATOR PENGELOLA MAGANG
2.MAHASISWA DITERIMA UNTUK MENDAPATKAN SURAT PENGANTAR MAGANG ATAU (MAHASISWA MENGINPUTKAN SURAT TERIMA DARI PERUSAHAAN(OPSIONAL))
3.MENGINPUTKAN LINK DRIVE LOGBOOK BULAN PERTAMA PADA DOSEN PEMBIMBING LAPANGAN DAN KOORDINATOR PENGELOLA MAGANG
4.MENGINPUTKAN LINK DRIVE LOGBOOK BULAN KEDUA PADA DOSEN PEMBIMBING LAPANGAN DAN KOORDINATOR PENGELOLA MAGANG
5.MENGINPUTKAN LINK DRIVE LOGBOOK BULAN KETIGA PADA DOSEN PEMBIMBING LAPANGAN DAN KOORDINATOR PENGELOLA MAGANG
6.MAHASISWA MENGINPUTKAN LAPORAN UNTUK REVIEW DOSEN PEMBIMBING LAPANGAN
7.MAHASISWA MENGAJUKAN DESIMINASI INPUT LAPORAN HASIL MAGANG BERDASARKAN ACC DARI DOSEN PEMBIMBING LAPANGAN
8.MAHASISWA MENDAPATKAN JADWAL DESIMINASI DAN PENGUJI DESIMINASI
9.MAHASISWA MELAKUKAN DESIMINASI
10.MAHASISWA MELAKUKAN REVISI DARI PENGUJI DESIMINASI
11.MAHASISWA MELAKUKAN INPUT LAPORAN HASIL MAGANG AKHIR
12.MAHASISWA MELIHAT HASIL DESIMINASI MAGANG


NOTE: SIFAT PROPOSAL ADALAH MENUJUKKAN AKTIVITAS SELAMA MAGANG GUNA MELAPORKAN HASIL MAGANG DENGAN BENTUK LAPORAN AKHIR MAGANG
